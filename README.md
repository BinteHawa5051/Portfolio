# Tooba Akram — Portfolio

Personal portfolio website for Tooba Akram, AI/ML Engineer and Software Engineering student at NUST, Rawalpindi, Pakistan.

🔗 **Live:** [https://portfolio-zeta-woad-93.vercel.app](https://portfolio-zeta-woad-93.vercel.app)

---

## Features

- Interactive particle background — 220 glowing dots that repel away from the cursor
- Glassmorphism cards with backdrop blur
- Smooth scroll navigation with active link highlighting
- Mobile responsive with hamburger menu
- Contact form with mailto integration
- Deployed on Vercel with auto-deploy on push

## Sections

- **Hero** — Name, role, education (NUST, CGPA 3.55), GitHub & LinkedIn CTAs
- **Projects** — Intelligent Health Monitoring System, Academix, ML Internship Projects
- **Experience** — Elevo Pathways (AI/ML Intern), Alkhidmat Trust Foundation (Youth Intern)
- **Skills** — Python, TensorFlow, LangChain, FastAPI, PostgreSQL, XGBoost, and more
- **Contact** — Email, LinkedIn, GitHub + message form

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no frameworks, no build tools
- Canvas 2D API for the particle system
- Google Fonts (Space Grotesk, Inter, Fira Code)
- Deployed on Vercel

## Run Locally

Just open `index.html` in any browser — no server or install needed.

```bash
git clone https://github.com/BinteHawa5051/Portfolio.git
cd Portfolio
# open index.html in your browser
```
